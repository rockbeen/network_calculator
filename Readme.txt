Задание:
Написать простые программы, демонстрирующую механизм межпроцессного взаимодействия.
Требуется написать две программы, одна из которых занимается вычислениями (сложение,
вычитание, умножение, деление чисел), а другая принимает запросы от пользователя (без
разницы, каким образом - ожидает ввода от пользователя для консольного приложения, читает
значения из элементов формы для приложения с графическим элементом пользователя или с вебстраницы и т.п.), после этого второе приложение отправляет операнды и тип операции первому
приложению и ждет ответа. Первое приложение, приняв запрос, вычисляет результат и
отправляет его второму приложению, которое, в свою очередь, отображает результат
пользователю
